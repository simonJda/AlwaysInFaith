Privacy Policy

We take the protection of your personal data very seriously. This Privacy Policy explains what data is collected when you visit this website, how it is used, and what rights you have under the General Data Protection Regulation (GDPR).

1. Controller (Responsible party)

Simon Jarda

The controller is the natural person responsible for data processing on this website within the meaning of Art. 4 (7) GDPR.

2. Server log files

When you visit this website, the server automatically collects and stores information in so-called server log files. These include:

  • IP address
  • Date and time of the request
  • Requested URL
  • HTTP status code
  • User agent (browser and operating system)

This data is processed to ensure the technical operation, stability, and security of the website. The legal basis is Art. 6 (1) lit. f GDPR (legitimate interest).

3. Login and security-related data

When using the login functionality, the following data may be processed:

  • IP address
  • Number of login attempts
  • Timestamps of login attempts

This processing is carried out exclusively for security purposes, in particular to detect and prevent unauthorized access and brute-force attacks. The legal basis is Art. 6 (1) lit. f GDPR.

4. Data storage duration

Personal data is stored only for as long as necessary to achieve the respective purpose. Security-related data (e.g. login attempts) is automatically deleted or reset after a short period (for example a few hours).

5. Hosting

This website is hosted on a virtual private server (VPS). All data processing takes place on servers used exclusively for operating this website. No data is sold or passed on to third parties.

6. SSL / TLS encryption

This website uses SSL/TLS encryption to protect transmitted data. You can recognize an encrypted connection by the "https://" prefix in the address bar.

7. Your rights

You have the following rights under the GDPR:

  • Right of access (Art. 15 GDPR)
  • Right to rectification (Art. 16 GDPR)
  • Right to erasure (Art. 17 GDPR)
  • Right to restriction of processing (Art. 18 GDPR)
  • Right to object to processing (Art. 21 GDPR)
  • Right to lodge a complaint with a supervisory authority
